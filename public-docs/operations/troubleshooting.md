# Troubleshooting

## Science Query Returns Only Web Context

Check classifier settings, LLM logs, provider keys, quotas, and cooldowns.

## Provider Not Called

Possible causes: not in provider order, cooling down, quota reached, missing key, or query classified as non-science.

## 429

Quota or throttling. Check local usage and provider dashboard.

## 402

Payment, subscription, or account issue.

## Empty Extraction

Possible causes: bot blocking, scanned PDF, unsupported content type, timeout, publisher access controls, or wrong URL.

## Logs Empty

Check log paths, permissions, auth, and service user.

# Support

Searchbox is open-source software. Public support happens through GitHub issues and discussions when enabled.

## Before Opening An Issue

- Check the README and `public-docs/`.
- Check `/health` and `/health/monitor`.
- Review provider event logs and LLM attempt logs.
- Remove all API keys and private URLs from pasted output.

## Security Issues

Do not open a public issue for a vulnerability. See [SECURITY.md](SECURITY.md).

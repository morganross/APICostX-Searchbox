# Code of Conduct

Searchbox uses the Contributor Covenant Code of Conduct.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but accept corrections.
- Keep technical disagreements focused on the work.
- Help make the project understandable for new contributors.

## Unacceptable Behavior

Harassment, threats, personal attacks, discriminatory language, and sustained disruption are not acceptable.

## Enforcement

Maintainers may remove comments, close issues, block users, or take other appropriate action to protect the project community.
